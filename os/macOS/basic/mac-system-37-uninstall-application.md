- 安装软件
  - 打开 App Stroe 从官方地方下载安装，跟 iOS 手机系统一样道理
  - 从网络上下载安装，必须在安全性设置中解除来源问题，具体看我另外一篇文章。
  - 从网络上下载下来的文件又分为几种，一种是如下图这种直接拖到 Application 标记上即可的。
  - 还有一种是类似 Windows 的软件，有下一步下一步的，比如 Mac 下的 Oracle JDK 安装就是这样的。
